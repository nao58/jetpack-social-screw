Jetpack Social-Screw

これは、現在閲覧中のページのブックマーク数を表示する、非公式Jetpack Featureです。対応SBSは、現在のところHatenaとLivedoorだけです。

This is unofficial Jetpack Feature which display numbers of users who bookmarked on current page. The SBS services are only Hatena and Livedoor so far.

当然ながら、まだ入っていない場合はまず、Jetpackのインストールが必要です。

Of course, you'll need to install Jetpack if you haven't already.
